## 📋 About this Application

This software classifies music into genres using sound analysis. It uses deep learning and processes sounds as mel-spectrogram images. The goal is to help you identify music genres with about 90% accuracy. You do not need any technical skills to use this program.

The program works on Windows computers. It reads audio files, analyzes them, and tells you the genre. You can use this for personal projects, learning, or music sorting.

## 💻 System Requirements

To run this software, your Windows PC should meet these needs:

- Operating system: Windows 10 or newer (64-bit recommended)  
- RAM: 8 GB or more  
- Storage: At least 500 MB free disk space  
- Processor: Intel Core i3 or better / equivalent AMD  
- Internet connection: Required for the initial download and setup  
- Sound card and speakers or headphones (for audio playback if needed)

No additional hardware is needed.

## 🎯 Features

- Classify music genres such as rock, jazz, pop, classical, and more  
- Uses mel-spectrograms for accurate audio representation  
- Based on a Convolutional Neural Network (CNN) model trained on GTZAN dataset  
- Simple user interface to load audio and see results  
- Supports common audio formats like MP3, WAV, and FLAC  
- Runs locally on your Windows PC without sending data online
